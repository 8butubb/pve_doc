# 铭凡 N5 Air

铭凡 N5 Air 是铭凡推出的一款高性能 NAS 迷你主机，搭载 AMD Ryzen 7 255 处理器，兼顾家庭数据中心与轻量级生产力工具的双重需求。

## 硬件规格

| 项目 | 规格 |
|------|------|
| 处理器 | AMD Ryzen 7 255，8核16线程，基础频率 3.8GHz，加速频率 4.95GHz，TDP 45W |
| 核显 | AMD Radeon 780M（RDNA 3.0 架构，4nm 工艺） |
| 内存 | 双通道 DDR5 5600MHz，最高支持 96GB（非 ECC） |
| 系统盘 | 预装 64GB SSD（MinisCloud OS 专用） |
| M.2 插槽 | 1× M.2 2230/2280/22110（PCIe 4.0 x1，最高 4TB） |
| U.2/M.2 插槽 | 1× U.2 或 M.2 2280/22110（PCIe 4.0 x2，最高 15TB） |
| SATA 硬盘位 | 5× 3.5英寸/2.5英寸（单盘最高 22TB） |
| 网口 | 1× 10GbE（AQC113）+ 1× 5GbE（RTL8126） |
| 无线 | Wi-Fi 6E + Bluetooth 5.2 |
| OCuLink | 支持 eGPU 扩展 |
| PCIe 扩展 | 1× 半高 PCIe x16 插槽（PCIe 4.0 x4） |
| 前置接口 | 1× USB4（40Gbps，支持 DP）、1× USB 3.2 Gen2 |
| 后置接口 | 1× USB4（40Gbps，支持 DP）、1× HDMI 2.1、1× OCuLink、1× USB 3.2 Gen2、1× USB 2.0 |
| 散热 | 双 90mm 风扇 + 独立涡轮风扇（CPU/内存/M.2 定向散热） |
| 电源 | DC 5525，19V 14.73A，280W |
| 尺寸 | 199 × 202 × 252 mm |
| 重量 | 约 4kg |
