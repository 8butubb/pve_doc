# 我的需求

## 硬件要求
- **性能强劲**：能够稳定运行多个虚拟机
- **体积小巧**：替代现有体积过大的DIY主机

## 虚拟化规划

### LXC 容器（3个）
| 用途 | 说明 |
|------|------|
| 服务容器 | 运行各类服务 |
| 开发容器 | 开发环境 |
| 网关容器 | 网络网关 |

### 虚拟机（3个）
| 系统 | 用途 | 备注 |
|------|------|------|
| DSM | 主NAS系统 | 核心存储服务 |
| Windows 10 | 日常使用 | 计划直通RTX 3060显卡 |
| FNOS | 尝鲜体验 | 轻度使用 |

## 扩展计划
- 后期通过PCIe HBA卡连接硬盘柜后使用TrueNAS作为冷备份方案
