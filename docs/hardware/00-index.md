# 宿主机硬件选择

## 铭凡 N5 Air

### 硬件规格

| 项目 | 规格 |
|------|------|
| 处理器 | AMD Ryzen 7 255，8核16线程，基础频率 3.8GHz，加速频率 4.95GHz，TDP 45W |
| 核显 | AMD Radeon 780M（RDNA 3.0 架构，4nm 工艺） |
| 内存 | 双通道 DDR5 5600MHz，最高支持 96GB（非 ECC） |
| M.2 插槽 | 2× M.2 + 1× M.2 2230/2280/22110（PCIe 4.0 x1，最高 4TB） |
| U.2 插槽 | 最多 2× U.2（需扩展卡，PCIe 4.0 x2，最高 15TB），与 M.2 插槽共享位置 |
| SATA 硬盘位 | 5× 3.5英寸/2.5英寸（单盘最高 22TB，JMB585 桥接芯片） |
| 网口 | 1× 10GbE（RTL8127）+ 1× 5GbE（RTL8126） |
| OCuLink | 支持 GPU 扩展(PCIe 4.0 x4) |
| PCIe 扩展 | 1× 半高 PCIe x16 插槽（PCIe 4.0 x4） |
| 前置接口 | 电源键、状态指示灯、1× USB4（40Gbps，支持 DP）、1× USB 3.2 Gen2 |
| 后置接口 | 防盗锁孔、1× USB4（40Gbps，支持 DP）、复位键、1× HDMI 2.1、1× OCuLink、1× USB 3.2 Gen2、1× 5GbE、1× 10GbE、1× USB 2.0、电源接口 |
| 内部接口 | 预留 USB 接口（可接系统启动 U 盘） |
| SATA 控制器 | JMB585（PCIe 3.0 x2 转 5× SATA 3.0） |
| 散热 | 双 90×90×25mm 风扇 + 独立涡轮风扇（CPU/内存/M.2 定向散热） |
| 机身材质 | 塑料（N5 为铝合金） |
| 电源 | DC 5525，19V 14.73A，280W |


附上[电子说明书](https://pc-file-web.oss-cn-shenzhen.aliyuncs.com/N5%20%20N5PRO/user%20manual/F8NAA-%E7%94%A8%E6%88%B7%E6%89%8B%E5%86%8C-%E7%AE%80%E4%BD%93%E4%B8%AD%E6%96%87-V2.0_compressed.pdf)

该产品的优势在于高度集成、性能强劲且体积小巧
主要看中他的扩展性，支持PCIe 4.0 x4扩展，OCuLink扩展还有双usb4接口
